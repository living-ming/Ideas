# Ideas - WinUI 3 个人笔记工具
一个基于 WinUI 3 构建的高效笔记工具，支持注释管理、日程安排、日历联动、自定义背景主题。
## ✨ 功能特色

- 📝 笔记记录与格式化编辑
- 📌 右键注释与边栏管理
- 📅 日历与日程事件联动
- 🎨 自定义背景
